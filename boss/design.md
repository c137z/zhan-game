## Boss 标准接口

```js
{
  id: 'string',          // 唯一标识（snake_case）
  name: 'string',        // 中文名
  emoji: 'string',       // UI 表情
  desc: 'string',        // 描述/机制说明
  maxHP: number,         // 最大生命值
  baseAtk: number,       // 基础攻击力
  powerGrowth: number,   // 每回合攻击力成长（可省，默认 0）
  startShield: number,   // 初始护盾
  cycle: [{              // 行动循环
    type: 'attack'|'defend'|'focus'|'crit'|'stun',
    shield: number       // defend 类型的护盾值
  }],
  traits: [{             // 特殊机制
    id: 'string',
    events: ['TURN_START'|'TURN_END'|'RESOLVE'],
    params: {}
  }],
  hpTriggers: ['groom'|'hiss'|...],  // 血量触发事件
}
```

## 猫咪冒险 — 关卡通用结构

### 通用流程

```
进图 → 二选一圣物（分享刷新 / 广告全拿）
     → 进入战斗
     → 打赢 Boss
     → 通关结算（+5 猫毛）
     → 弹出：继续闯关 / 返回主页
```

### 通用规则

- **圣物**：进图二选一，附"分享刷新"和"广告全拿"按钮
- **猫毛**：通关固定 5，不再单独说明
- **玩家**：HP 100，10 格消除槽，5×5 牌堆
- **普通关卡无成长压力**：powerGrowth 默认 0，老大说加再加
- **胜利条件**：Boss HP ≤ 0 或牌库清空（元气弹斩杀）

### 设计协议

以后设计新关卡只需三个字段：

| 字段 | 说明 |
|------|------|
| **Boss** | 谁 + 行为逻辑 |
| **卡牌** | 种类 × 数量 |
| **目的** | 教学什么 |

通用流程和猫毛自动对齐，不用重复说。

## 关卡 1：教学关 — 逗猫棒 🪄

认识游戏——了解消牌/连击/攻防基础玩法，建立决策自信。

| 字段 | 值 |
|------|-----|
| **Boss** | 逗猫棒 🪄（`catToy`），HP 50，攻 5 / 防 5(盾) 交替 |
| **卡牌** | 攻击 🗡 50 + 防御 🛡 50，共 100 张 |
| **目的** | 玩家轻松做出 3-5 连的正确选择 → 炫酷动画反馈 → "我好聪明" |

**Boss 实现**：`code/data.js` `BOSSES.catToy` ✅ 已有

## 关卡 2：增伤入门 — 水杯 🥤

一碰就碎，引入暴击和破甲——让玩家感受到伤害翻倍的爽感。

| 字段 | 值 |
|------|-----|
| **Boss** | 水杯 🥤，HP 100，攻 10 / 防 10(盾) 交替 |
| **卡牌** | 攻击 50 + 防御 50 + 暴击(攻击加成 ⚡)50 + 破甲(易伤 💔)50，共 200 张 |
| **目的** | 认识增伤体系：暴击 ×1.5 + 破甲 ×1.5 → 基础伤害翻倍以上，体验秒杀的快感 |

## 关卡 3：防守入门 — 小蜜蜂 🐝

尾针扎猫很痛！引入虚弱和减伤——让玩家学会保护自己。

| 字段 | 值 |
|------|-----|
| **Boss** | 小蜜蜂 🐝，HP 100，攻 20 / 防 20(盾) 交替 |

## 关卡 4：进阶博弈 — 小蛇 🐍

伤害很高但可以控和苟。只有击晕、回血、万能三种控制/续航牌，没有攻击牌。6 回合控住打空牌库即可元气弹斩杀。

| 字段 | 值 |
|------|-----|
| **Boss** | 小蛇 🐍，HP 100，蓄力 → 攻 50 循环 |
| **卡牌** | 击晕(眩晕 💫)20 + 回血(❤️)20 + 万能(💎)20，共 60 张 |
| **目的** | 教控制+续航+灵活补位：晕掉高攻回合 → 回血苟住 → 万能补连击 → 元气弹灭杀 |

### 6 回合教科书

| 回合 | 小蛇 | 玩家 |
|:---:|------|------|
| 1 | 蓄力 | 输出（叠晕） |
| 2 | 攻 50 | **击晕跳过！** |
| 3 | 蓄力 | 输出 |
| 4 | 攻 50 | **击晕跳过！** |
| 5 | 蓄力 | 冲刺输出 |

## 猫猫迷宫（日常资源）

> 毛线团 → 抽圣物 → 随机猫 Boss 连续战。

### 通用规则

- **流程**：第一关毛线团 🧶 → 打完血量不变（不回满，剩多少就多少）→ 二选一圣物（分享刷新/广告全拿）→ 打随机猫 Boss
- **猫 Boss**：从 10 只猫 Boss 中随机一只
- **卡牌**：标准卡池
- **奖励**：
  - 首次击败该猫 Boss：50 猫毛 + 猫猫形象奖杯 + 成就（每只猫仅一次，共 10 只 = 500）
  - 每日前 3 次通关：10 猫毛（最多 30/天）
  - 重复通关已击败过的猫：无奖励
- **通关弹窗**：再次挑战 / 回到主页
- **无尽模式**：无

### 猫毛获取

| 来源 | 数量 | 说明 |
|------|:---:|------|
| 冒险关卡通关 | 5/关 | 50 关全通 = 250 |
| 猫猫迷宫首杀 | 50/只 | 10 只猫 = 500 |
| 猫猫迷宫每日 | 10/次 | 每天前 3 次，最多 30/天 |
| 看广告 | 5/次 | 30 秒激励广告 |

> 猫毛消耗（局外成长）待后续设计。

## 猫王塔（无尽挑战）

> 超级困难。必须了解全部猫猫特性 + 好圣物 + 好运气才能打通。

### 规则

- 进塔随机一个圣物（仅 1 个，无法选择）
- 用这一个圣物打败第一只随机猫 Boss
- 胜利后随机抽第二个圣物（也是最后一个，共 2 个圣物）
- 带着 2 个圣物一层层打随机猫 Boss，**每层之间血量状态重置**
- 全部 10 只猫 Boss 打完才算通关
- 预计耗时：1 小时+
- **排行规则**：通关按总回合数排名。回合最少者为宇宙猫王，其余依次降级

### 地位划分

```
第 1 层：社区猫王 — 击败 1 只
第 2 层：街道猫王 — 击败 2 只
第 3 层：城区猫王 — 击败 3 只
第 4 层：城市猫王 — 击败 4 只
第 5 层：省会猫王 — 击败 5 只
第 6 层：大区猫王 — 击败 6 只
第 7 层：王国猫王 — 击败 7 只
第 8 层：大陆猫王 — 击败 8 只
第 9 层：星球猫王 — 击败 9 只
第 10 层：宇宙猫王 🌌 — 击败 10 只（全通）
```

> 每击败一只自动晋升，弹窗显示新称号。全通显示「宇宙猫王」——全球排行展示。
