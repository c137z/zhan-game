# 主界面 UI 规格

> 复古像素风 · 单列纵向居中 · 三板块结构

---

## 整体设定

| 项目 | 值 |
|------|-----|
| 视觉风格 | 复古像素风（字体/图标/边框带像素颗粒感+深色描边） |
| 背景色 | `#1E222D`（深蓝灰） |
| 布局 | 单列纵向居中，两条横向虚线分割线分三区 |

---

## 第一区：标题 + 今日状态

### 主标题
- 单个大字："斩"
- 白色像素字体，右下黑色像素阴影
- 居中靠顶

### 分割线
- 浅灰色像素点组成的横向虚线

### 今日状态
- "今日最佳：62层"
- 白色像素字体，小字号
- 居中

---

## 第二区：三大玩法入口

三个垂直排列的圆角矩形大长条按钮。每个：左侧图标 + 右侧上下两行文字。

### 按钮一：猫咪冒险（主线关卡）

| 属性 | 值 |
|------|-----|
| 背景色 | 灰蓝色 + 深色像素内边框 |
| 图标 | 像素风三花/黄白相间猫咪头像（白胡须+深色耳朵） |
| 标题 | **猫咪冒险**（白色粗体） |
| 副标题 | 主线关卡（浅灰色，小字号） |

### 按钮二：猫猫迷宫（每日资源）

| 属性 | 值 |
|------|-----|
| 背景色 | 墨绿色 + 深色像素内边框 |
| 图标 | 正方形浅灰色像素迷宫图案（中心有'G'字形结构） |
| 标题 | **猫猫迷宫**（白色粗体） |
| 副标题 | 每日资源（浅灰色，小字号） |

### 按钮三：猫王塔（无尽挑战）

| 属性 | 值 |
|------|-----|
| 背景色 | 红褐色/暗棕色 + 深色像素内边框 |
| 图标 | 像素风灰色城堡/防御塔，塔顶金色皇冠 |
| 标题 | **猫王塔**（白色粗体） |
| 副标题 | 无尽挑战（浅灰色，小字号） |

---

## 第三区：资产 + 系统功能

### 分割线
- 与第一道相同的浅灰色像素横向虚线

### 资产显示
- 左侧：黄褐色毛茸茸像素猫毛图标
- 右侧：白色文字 "猫毛：1250"

### 系统功能（2×3 九宫格）

6 个正方形圆角功能按钮，深红褐色底色 + 深色边框，下方白色括号标签。

| 位置 | 名称 | 图标 |
|:---:|------|------|
| 一排左 | 每日悬赏 | 红白射箭靶盘，中心插绿尾黑箭 |
| 一排中 | 图鉴 | 闭合暗棕色硬皮书，封面金色方块 |
| 一排右 | 成就 | 金色圆形奖牌，红蓝像素缎带 |
| 二排左 | 好友排行 | 两只小猫头像（左黄右灰），黄猫头顶金皇冠 |
| 二排中 | 猫王地图 | 展开的土黄色羊皮纸地图，点状路线+标志 |
| 二排右 | 设置 | 经典浅灰色像素齿轮 |
