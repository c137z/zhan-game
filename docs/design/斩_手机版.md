# 「斩」— 回合制堆叠消除·牌组构筑游戏

> 游戏设计文档 v0.1 | 构思：老大 + 小龙虾 🦞 | 灵感：杀戮尖塔 + 吸血鬼幸存者 + 羊了个羊

---

## 一、游戏概念

**堆叠消除 + 回合制构筑 + roguelike爬塔——消除即是蓄力，出招即是斩杀。**

### 核心循环

爬塔 → 遭遇敌人 → 回合制对局：
① 从堆叠牌堆中点击选中可见牌
② 按顺序放入 10 格消除槽
③ 可提前结束回合（不满 10 格也行）
④ 自动结算：扫描槽内连续相同元素（≥3）→ 生成对应属性
⑤ 所有牌消耗 → 清空槽 → 进入下一回合
⑥ 循环直至一方血量为零
胜利 → 奖励选择 | 失败 → 局外解锁

## 二、战斗系统（核心）

### 2.1 牌堆

- 固定牌堆：每场战斗开局固定生成，**不会刷新或补充**
- 堆叠机制：上层牌遮住下层牌
- 只能点击当前可见牌
- 牌的数量和构成由关卡难度、精英状态、遗物等决定

### 2.2 消除槽（10 格）

固定 10 格，点击桌面上的可见牌 → 按点击顺序进入槽位（从左到右）。
点击顺序不可调整。可提前结束回合（不满 10 格也行），但已放入的牌不可撤回。

回合结束时自动结算：
- 扫描槽内连续相同元素（必须相邻且同色）
- 连续 ≥ 3 个 → 按连击表计算对应属性
- **所有槽内牌（包括未合成的）全部消耗掉**
- 清空槽位，进入下一回合

### 2.3 核心策略

**看牌：** 点走上层牌后下层暴露。即使这回合没有合成，看清下层分布就是巨大收益。

**垫刀：** 前几回合点走杂牌暴露下方核心组合。若垫刀中凑成连击，还能顺带获得属性加成。

### 2.4 连击结算表（示例数值）

3连：攻击10 / 防御+5盾 / 治疗+3血
4连：15 / +8盾 / +5血
5连：25 / +13盾 / +8血
6连：40 / +20盾 / +12血
7连：65 / +30盾 / +18血
8连：100 / +45盾 / +28血
9连：150 / +65盾 / +40血
10连：250 / +100盾 / +60血

设计原则：连击数越高呈指数增长，鼓励攒大招。

### 2.5 牌的类型

🔴 红（攻击）→ 连击造成伤害
🔵 蓝（防御）→ 连击叠加护盾
🟢 绿（治疗）→ 连击回复生命
🟡 金（万能）→ 可充当任意颜色参与连击（不能单独形成连击）
⚫ 黑（诅咒）→ 占位废牌，消耗不产生效果

## 三、战斗流程

① 玩家回合：观察可见牌堆 → 决策（攒输出/堆防御/垫刀看牌）→ 点击进槽 → 结束回合
② 回合结算：扫描连击 → 所有牌消耗 → 属性生效
③ 敌人回合：执行行动模式 → 攻击/塞诅咒 → 回到①

## 四、爬塔框架（后续扩展）

### 岔路节点
⚔️ 普通战 — 标准敌人，少量奖励
💀 精英战 — 强敌+更多诅咒牌，高奖励
🏪 商店 — 花钱买新牌/遗物
🔥 休息 — 回复30%生命
🎁 宝箱 — 免费稀有牌/遗物

### 遗物示例
磨刀石：红色连击伤害+20%
铁壁：每回合自动获得3盾
虹光镜：万能牌参与的组合+1连击计数
诅咒玉佩：消耗黑色牌时回复2血
贪婪之眼：预知下一层牌面

## 八、设计理念

1. 固定牌堆 = 每张牌都珍贵
2. 视野即是收益
3. 垫刀 = 资源管理
4. 爆发节奏 — 布局几回合藏大招
5. 运气与策略并存
6. 永不刷新 — 有限资源下的深层博弈
