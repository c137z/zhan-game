## 一、游戏概念

**堆叠消除 + 回合制构筑 + roguelike 爬塔——消除即是蓄力，出招即是斩杀。**

### 核心循环

爬塔 → 遭遇敌人 → 回合制对局：
① 从堆叠牌堆中点击选中可见牌
② 按顺序放入消除槽，点击顺序不可调整
③ 可提前结束回合，已放入的牌不可撤回
④ 自动结算：扫描槽内连续相同元素（≥3）→ 多段并行结算
⑤ 所有牌消耗 → 清空槽 → 进入下一回合
⑥ 循环直至一方血量为零
胜利 → 奖励选择
失败 → 本次爬塔结束

## 二、战斗系统（核心）

### 2.1 牌堆

- **牌库规模**：白盒原型 135 张（红 50 + 蓝 50 + 绿 35），用完即止
- 堆叠机制：3×3 展板，上层遮下层，只能点击当前可见牌
- 每摞牌厚度：15 张 → 3 层（3×3 表面可见，下层 3×3 部分遮挡）

### 2.2 消除槽（白盒原型 6 格）

固定 6 格，牌放入后不可撤回，顺序从左到右固定。

#### 进槽操作（移动端优化）

**点选 + 确认吸入**：
1. 单指点击牌堆中的牌 → 牌飞出半格悬浮，消除槽对应位置显示**虚线预览位**
2. 再次点击（或点击消除槽中对应的虚线位）→ 牌吸入槽内
3. 替代双击，避免 iOS Safari 缩放冲突

#### 即时预览

消除槽上方显示**当前槽位连击预览**，随牌加入动态更新。

#### 结算规则

- 扫描槽内连续相同元素（必须相邻且同色），连续 ≥ 3 个 → 按连击表计算
- **多段并行结算**：同时识别所有连击组，各自计算效果
- **所有槽内牌（包括未合成的单牌）全部消耗掉**

#### 未消除惩罚（白盒原型）

回合结束时，槽内**未形成连击的牌**每张扣玩家 **1 HP**。防止无脑过牌。

### 2.3 核心策略

| 策略 | 说明 |
|------|------|
| **看牌** | 点走上层牌后下层暴露。即使没合成，看清下层分布也是收益 |
| **垫刀** | 故意点走杂牌暴露下方好牌，顺便可能凑出连击 |
| **憋大招** | 布局几回合攒同色，打出高连击一波带走 |
| **空库流** | 牌库全消直接获胜 |

## 三、敌人系统

> 防止玩家无限憋 6 连，必须有行动压力逼玩家做取舍。

### 3.1 白盒敌人属性

| 属性 | 数值 |
|:----:|:----:|
| HP | 100 |
| 初始护盾 | 0 |
| 基础攻击 | 12 |
| 力量增长 | 每回合 +2 |
| 狂暴 | 每 5 回合攻击×2 |

### 3.2 玩家初始属性

| 属性 | 数值 |
|:----:|:----:|
| HP | 100 |
| 护盾 | 0 |

### 3.3 胜利/失败条件

- 胜利：牌库全消 或 敌人 HP ≤ 0
- 失败：玩家 HP ≤ 0

### 3.4 固定行为循环

| 回合 | 行为 |
|:---:|:----:|
| 1 | 攻击 |
| 2 | 防御 |
| 3 | +力量 |
| 4 | 攻击 |
| 5 | 攻击 |
| 6 | 防御 |
| 7 | **狂暴（攻击×2）** |
| (8+) | 回到回合 1 循环 |

固定循环便于测试平衡——不会因为 AI 随机性干扰对玩家策略的观察。

### 3.5 战斗流程

```
① 消牌阶段：观察牌堆 → 点选进槽 → 结束回合
   勇者行动开始 → 清空勇者旧盾
② 勇者结算：扫描连击 → 多段并行 → ①回血→②加盾→③攻击→未消除惩罚 → 所有牌消耗
   幽灵行动开始 → 清空幽灵旧盾
③ 幽灵行动：执行固定行动（攻击/防御/加力量）——此时勇者新盾有效，可挡伤
④ 回合结束 → 回合计数器+1 → 回到①
```

**护盾生命周期**：
- 勇者加的盾：勇者行动时加上 → 持续到**下回合勇者行动开始**（正好挡幽灵整轮攻击）
- 幽灵加的盾：幽灵行动时加上 → 持续到**下回合幽灵行动开始**（挡勇者整轮攻击）
- 护盾在**各自行动开始前清空**，不能跨回合堆叠
- 类似杀戮尖塔的护甲机制

## 附录 A：设计理念

1. **信息层级随机性** — 不是抽牌随机，是翻牌随机，信息暴露本身就是资源
2. **视野即是收益** — 哪怕空过，看清牌堆就是赚
3. **垫刀 = 资源管理** — 用杂牌换信息暴露
4. **爆发节奏** — 布局蓄力，一波斩杀
5. **你选的 Build 决定你怕什么** — 颜色干旱从运气变为选择
6. **永不刷新（大牌堆）** — 牌库比例就是你的策略
7. **元气弹终斩** — 空库不是苟活，而是蓄力一击

```
普-普-奇遇-精英-普-普-商店-精英-休息-BOSS
```

| 层 | 类型 | 奖励 | 说明 |
|:--:|------|------|------|
| 1 | 普通怪 | 增删卡 + 10金币 | 热身 |
| 2 | 普通怪 | 增删卡 + 10金币 | 积累 |
| 3 | 奇遇 | 见7.3表 | 变数 |
| 4 | 精英怪 | 增删卡 + 20金币 + 遗物 | 首个遗物 |
| 5 | 普通怪 | 增删卡 + 10金币 | |
| 6 | 普通怪 | 增删卡 + 10金币 | |
| 7 | 商店 | 消费 | 金币峰值：60 |
| 8 | 精英怪 | 增删卡 + 20金币 + 遗物 | 第二个遗物 |
| 9 | 休息 | 回血30 或 增强牌组 | 决战前整备 |

**战斗统计**：4普 + 2精英 + 1BOSS = 7场（奇遇可能+1场小怪/精英）

**时间估算**：
- 前期普通怪：3-4分/场
- 精英/BOSS：稍长
- 总时长：约30-40分钟一局

### 7.2 金币经济

| 收入来源 | 金币 |
|----------|:---:|
| 普通怪 ×4 | 40 |
| 精英怪 ×2 | 40 |
| 奇遇 | 0-20 |
| **到达商店时合计** | **60-80** |

**商店定价**（60金币可买2-3件）：

| 商品 | 单价 | 效果 |
|------|:---:|------|
| 增卡 5张 | 20 | 随机三色三选一 |
| 删卡 5张 | 20 | 随机三色三选一 |
| 大幅增卡 10张 | 40 | 随机三色三选一 |
| 大幅删卡 10张 | 40 | 随机三色三选一 |
| 随机遗物 | 40 | 二选一 |
| 回血 10点 | 20 | — |
| 生命上限 +10 | 40 | 长期价值 |

### 7.3 奇遇事件（10种，随机触发）

| 概率 | 事件 | 效果 |
|:---:|------|------|
| 30% | 增删卡 | 三选一增卡5张 + 三选一删卡5张 |
| 30% | 强制塞卡 | 随机塞入牌库5张（普通类型之一，非诅咒）|
| 10% | 打小怪 | 战斗胜利后增删卡 + 10金币 |
| 10% | 打精英 | 战斗胜利后增删卡 + 20金币 + 遗物 |
| 10% | 回血 | 直接回复10点HP |
| 10% | 扣血 | 损失10点HP |

**奇遇塞卡的设计理念**：
- 塞入的是普通类型卡（攻击/防御/回血等），不是诅咒
- **不是纯负收益**——增加了牌库厚度，但稀释了build浓度
- 配合增删卡机制：奇数塞卡→偶数删卡，牌库厚度在"膨胀vs收缩"之间动态平衡
- 玩法含义：打薄build追求元气弹 vs 打厚build追求combo稳定性

### 7.4 休息站

决战前唯一休息点，**二选一**：

| 选择 | 效果 | 适用场景 |
|------|------|---------|
| 回血30 | 直接回复30HP | 血量危险、稳妥通关 |
| 增强牌组 | 选一种类型增强 | 血量充足、追求爆发 |

**增强牌组效果**：
- 该类牌连击效果 ×1.5（攻击/防御/回血）
- Buff/Debuff 门槛降低：`3→2, 5→4, 6→5, 7→6` 变成 `2→1, 4→2, 5→3, 6→4`
- 万能牌 → 一张顶两张

**设计意图**："没血但想拼一波"vs"稳一手回血"——玩家会记住这种高张力选择。
增强后"激动的心颤抖的手，一鼓作气斩BOSS"是核心爽点。

### 7.5 增删卡机制

**所有增删卡都是随机三选一，不一张张选**：

| 操作 | 方式 | 来源 |
|------|------|------|
| 增卡 5张 | 随机三色 → 选一种加入5张 | 普通怪/精英怪/商店 |
| 删卡 5张 | 随机三色 → 选一种删去5张 | 普通怪/精英怪/商店 |
| 大幅增卡 10张 | 随机三色 → 选一种加入10张 | 商店 |
| 大幅删卡 10张 | 随机三色 → 选一种删去10张 | 商店 |

**设计意图**：
- 不确定性驱动不同build方向（想走攻击流但随机出防御/回血/万能，必须重新思考）
- 增卡→该类连击概率↑，删卡→连击概率↓
- BOSS战核心矛盾：**薄牌库（离元气弹近但输出弱）vs 厚牌库（输出稳但离元气弹远）**
- BOSS设计需要让纯憋元气弹几乎不可能——单回合输出不够就会被压制

### 7.6 敌人难度曲线

普通怪 → 精英怪 → BOSS 的数值递增设计：

| 类型 | HP | 基础攻击 | 特点 |
|------|:--:|:-------:|------|
| 普通怪 | 100 | 12 | 7回合固定循环 |
| 精英怪 | 200 | 18 | 加新机制（塞牌/破盾/debuff）|
| BOSS | 待定 | 待定 | 多阶段，强力机制 |
